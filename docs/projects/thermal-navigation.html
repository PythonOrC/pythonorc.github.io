Thermal Autonomous Navigation

Caltech ARCL • SURF Fellow Feb 2025 – Present
Hongyi Hu — Autonomous Robotics and Control Lab, Caltech
Robotics Perception Active Research

Description

Investigating map-free thermal navigation for aerial robots in low-light, over-water environments using imitation learning from traditional planners. The project targets a key research question: given that standard RGB pipelines do not provide desirable performance on thermal images, how can we design an image-to-action model trained on and designed for thermal input?

Although the project targets aerial robots, I currently train and iterate on a remote car platform outdoors to support rapid data collection and systematic debugging while preserving the core challenges of real low-light navigation: changing illumination, texture ambiguity, and the need for robust state estimation from imperfect observations.

The work spans the full pipeline: designing the learning model, collecting multimodal data (thermal, LiDAR, IMU), running traditional planners to generate supervision for imitation learning, and training models that accept different sensor inputs with modality switching.

Key Highlights

  • Awarded Caltech Class of '52 SURF Fellowship (July 2025).
  • End-to-end perception-to-action pipeline for thermal navigation.
  • Multimodal data collection: thermal, LiDAR, and IMU.
  • Imitation learning from traditional planner supervision.
  • Modality switching for degraded sensing conditions.
  • Long-term goal: thermal-only deployment for practical autonomy.
