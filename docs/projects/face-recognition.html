Face Recognition Door Unlock System

Personal Project 2025
Computer Vision ML

Description

A real-time face recognition system designed for door access control. The system captures webcam frames, assesses image quality (brightness, blur, lighting uniformity), detects and aligns faces, extracts embeddings, matches against a database, and performs anti-replay checks before granting access.

The architecture supports multiple interchangeable detection backends (dlib HOG, dlib CNN, MediaPipe, OpenCV DNN, YOLO) and recognition backends (dlib ResNet, DeepFace with ArcFace/Facenet, AdaFace). An ablation study framework is included for systematic comparison of backend combinations.

System Pipeline

  • Image Quality Check — Brightness, blur/sharpness, and lighting uniformity filtering.
  • Face Detection — Pluggable backends: dlib HOG/CNN, MediaPipe, OpenCV DNN, YOLO.
  • Preprocessing — Crop, pad/resize, and optional eye-based alignment.
  • Embedding Extraction — dlib ResNet (128-D), DeepFace (varies), or AdaFace (512-D).
  • Database Matching — Threshold-based with margin verification for unambiguous matches.
  • Anti-Replay — Frame hash comparison and reuse cooldown to detect spoofing.
  • Debug Panel — 3×3 grid showing intermediate processing images in real time.

Tech Stack

  • Python / OpenCV — Real-time webcam capture and visualization.
  • dlib — HOG/CNN face detection and ResNet embedding.
  • MediaPipe — Fast face detection backend.
  • DeepFace / AdaFace — State-of-the-art recognition models.
  • YOLOv8 — Object detection-based face finding.
  • NumPy — Embedding computation and distance metrics.
