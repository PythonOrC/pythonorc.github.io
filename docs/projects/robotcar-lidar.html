RobotCar LiDAR Vehicle Detection (2026)

Oxford RobotCar Dataset
Computer Vision LiDAR

Description

Extended the Oxford RobotCar Dataset SDK with vehicle-aware LiDAR projection. The system uses YOLOv8 segmentation to detect vehicles (cars, trucks, buses, motorcycles) in camera images, then intelligently downsamples the projected LiDAR point cloud: full resolution is preserved on detected vehicles while background points are reduced.

This is useful for autonomous driving applications where vehicle detail is critical but full point cloud density is unnecessary everywhere. The tool supports single-frame analysis, video playback with interactive controls, and video export.

Also developed a complementary vehicle segmentation pipeline using YOLO + SAM (Segment Anything Model) that produces segmented versions of the dataset imagery with transparent vehicle overlays.

Key Highlights

  • Vehicle detection using YOLOv8 segmentation on camera images.
  • Selective LiDAR downsampling: full resolution on vehicles, reduced background.
  • Video playback mode with pause, step-through, and export to MP4.
  • YOLO + SAM segmentation pipeline for dataset annotation.
  • Built on the Oxford RobotCar and Oxford Radar RobotCar datasets.

Tech Stack

  • Python / OpenCV / NumPy — Core processing and visualization.
  • YOLOv8 (Ultralytics) — Real-time vehicle segmentation.
  • SAM (Segment Anything) — Fine-grained vehicle segmentation.
  • Open3D — 3D point cloud visualization.
  • Oxford RobotCar SDK — Dataset loading, camera models, LiDAR processing.
