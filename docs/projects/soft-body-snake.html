Soft Body Robotic Snake

Genius Olympiad 2023 — Bronze Medal Aug 2022 – Jun 2023
Robotics Hardware

Abstract

Soft-body robots outperform traditional robotics in situations requiring great flexibility, with applications in medical, search-and-rescue, and environmental domains. This project created a multi-degree-of-freedom modular soft snake robot using pneumatically actuated silicone tubes arranged in a triangular pattern per segment, each providing 3 DoF. The robot implements serpentine and rectilinear locomotion controlled via sinusoidal gait functions distributed across independently controlled segments.

Simulia Abaqus FEA simulations were used to optimize silicone body parameters (wall thickness, inner diameter). The mold design went through four iterations to achieve reliable air-tight seals. Four wheel revisions improved speed by 331% (0.68 cm/s → 2.25 cm/s). The robot features ESP32-based distributed control with I2C communication, remote control via Wi-Fi/MQTT, and real-time video streaming from an ESP32-CAM module.

Key Highlights

  • Bronze Medal at the 2023 Genius Olympiad.
  • Modular design: each segment is independent with its own ESP32 controller and solenoid valves for pneumatic actuation.
  • Shore A-10 silicone rubber with 3 inflatable chambers per segment in a triangular pattern for 3-DoF bending.
  • Serpentine locomotion via distributed sinusoidal gait: k = sin(Speed × rads + id × shift).
  • FEA-optimized silicone parameters: 4mm wall thickness and 14mm diameter yield maximum bending angle.
  • Four mold iterations and four wheel revisions; final ratchet-style wheels achieved 331% speed improvement.
  • Remote control app built with MIT App Inventor; real-time video stream via ESP32-CAM HTTP server.

Tech Stack

  • Silicone casting (Shore A-10) — Custom 4-revision mold design for inflatable tube segments with air-tight seals.
  • Pneumatic actuation (12 kPa) — Solenoid valves controlling inflation/deflation of 3 chambers per segment.
  • ESP32 & ESP32-CAM — Distributed control via I2C bus, Wi-Fi/MQTT remote control, HTTP video stream.
  • Simulia Abaqus — FEA simulation for optimizing wall thickness and chamber diameter.
  • CNC routing & 3D printing — Acrylic structural plates (CNC), TPU tires and PLA mounts (3D printed).
  • Arduino — Firmware for ESP32 modules controlling gait sequencing and valve states.
