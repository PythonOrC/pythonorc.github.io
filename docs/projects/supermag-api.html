SuperMAG R API

Johns Hopkins University May – Jun 2022
Space Weather Software

Description

Wrote the R application programming interface (API) for Johns Hopkins University's SuperMAG project, a worldwide collaboration of organizations and national agencies that operate ground-based magnetometers. The API provides programmatic access to SuperMAG's extensive geomagnetic data archives, enabling researchers to download, process, and analyze magnetometer data directly from R.

The API has attracted approximately 4,000 registered users with over 5,000 downloads per month. The work was mentioned at the 2023 GEM Annual Conference, and I received a letter of acknowledgment from the project's Principal Investigator, Dr. Jesper Gjerloev.

Key Highlights

  • ~4,000 registered users and 5,000+ monthly downloads.
  • Mentioned at the 2023 GEM Annual Conference.
  • Letter of acknowledgment from PI Dr. Jesper Gjerloev.
  • Provides full programmatic access to the SuperMAG data archive.
